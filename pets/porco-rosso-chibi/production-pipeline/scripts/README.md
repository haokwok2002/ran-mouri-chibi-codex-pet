# 构建脚本预留

`prepare_reference.py` 将已有初始图隔离为统一纯绿幕身份参考图。它不调用图片服务，也不读取 `.env`；输出可作为后续在线图片生成与动作制作的第一参考图。

`normalize_chroma_background.py` 将图片服务返回的近似绿幕标准化为精确 `#00FF00`，只处理与画布边缘连通的绿色背景，不改写角色主体。

`generate_sequences.sh` 生成 8 组绿幕分镜；`build.sh` 统一去绿、逐格镜像左跑、构建 V1 图集并执行结构/质量检查。任何调用图片服务的脚本必须从工作区根目录共享 `.env` 读取配置，且不输出配置内容。
