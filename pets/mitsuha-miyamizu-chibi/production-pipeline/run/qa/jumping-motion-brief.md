# 三叶跳跃动作验收要点

三叶的 `jumping` 行固定为五帧，客户端播放速度不可由宠物素材调整。因此以清晰的关键姿势建立更从容、真实的节奏，而不通过提高跳跃高度或缩放角色来制造动感。

| 帧 | 动作节拍 | 必须读出的信息 |
| --- | --- | --- |
| 1 | 蓄力 | 双脚着地、轻蹲、重心下沉 |
| 2 | 轻离地 | 刚离地，身体仍略压缩 |
| 3 | 顶点 | 温和最高点，身体轻微舒展 |
| 4 | 回落 | 接近地面，膝盖开始准备缓冲 |
| 5 | 落地回稳 | 双脚回到基线、膝盖微屈，顺畅衔接第 1 帧 |

验收时特别检查：角色大小、头部高度比例与左右位置在五帧间稳定；仅以身体高度和姿态表现跳跃；编绳、流苏与裙摆只作轻微滞后。不得有悬浮感、突然弹跳、缩放闪烁、横向抖动、地面、投影、灰尘、落地标记、速度线或光晕。
