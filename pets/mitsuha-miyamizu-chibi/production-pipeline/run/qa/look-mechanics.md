# 三叶观察方向动作机制

三叶是站定的人形角色。观察方向不是把整个人旋转或倾斜，而是由眼神先行，头部、颈部与上半身以小幅度跟随；双脚、裙摆基线和整体身高保持稳定。

## 动作规律

- 眼睛：暖棕色眼睛与眼睑一起朝目标方向转动；不在固定眼白上滑动不自然的孤立瞳孔。
- 头部与躯干：眼神先走，头部以克制角度转向目标；越接近左右方向，近侧脸颊和眼睛略更可见，远侧眼睛略窄。上下方向通过眼睑、眉线、下巴与轻微低头/抬头表达。
- 编绳与侧马尾：随头肩方向轻微跟随，仍固定在发根；不得漂浮、换侧或跳跃。
- 手和服装：保持自然下垂或轻靠身体，领结、裙子和鞋的结构不变；不因观察方向新增道具或明显摆动。

## 四个方向锚点

| 方向 | 自然姿势 |
| --- | --- |
| `000` 上 | 眼睛与眉线向上，头略抬，下巴稍上扬，肩线不动。 |
| `090` 屏幕右 | 瞳孔、鼻尖、脸与头部向观者右侧转；右侧头发与编绳随头部轻微跟随。 |
| `180` 下 | 眼睛朝下，眼睑稍压，头轻低，脸仍完整可辨。 |
| `270` 屏幕左 | 瞳孔、鼻尖、脸与头部向观者左侧转；左侧面部更显，编绳维持与头部的自然相对位置。 |

## 连续性预算

每增加 `22.5°` 只增加近似同等幅度的眼神、头部和上身变化。`157.5 → 180`、`337.5 → 000` 必须像连续一小步，不允许换脸、换比例、换发绳位置、整体旋转、横向滑动或突然改表情。
