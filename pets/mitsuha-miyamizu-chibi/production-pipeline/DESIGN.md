# 三叶 · 紧凑 Q 版 Codex Pet 设计规范

## 身份与制作边界

- 约三头身的偏宽、偏矮 Q 版比例；暖棕大眼、深棕侧马尾、红色编绳与流苏、白衬衫红领结、深蓝百褶裙、深蓝袜和棕色乐福鞋是不可变身份锚点。
- 嘴部使用可在 `192×208` 宠物尺寸辨认的深棕短弧线：自然微笑时须有清楚的嘴线和轻微嘴角，不使用接近肤色的极细短点或模糊线条；表情仍保持克制、温柔。
- 每帧保持同一头身比例、脸部构造、服装和发绳；使用纯 `#00FF00` 绿幕，角色以外不出现地面、投影、光晕、灰尘、文字、边框、速度线或独立特效。
- 站定动作的脚底位置稳定；只允许跳跃和定向跑动出现合理的上下位移。不得以整体缩放、横向平移或旋转角色制造动作。
- 三叶的动作气质是温柔、认真、略带含蓄的青春感：靠眼神、头部、双手、编绳与裙摆的轻微二级运动表达，不做大幅夸张或机械弹跳。

## 九组标准动作分镜

| 状态 | 帧数 | 动作设计 | 节奏要点 |
| --- | ---: | --- | --- |
| `idle` | 6 | 安静呼吸与一次自然眨眼。 | 稳定站姿 → 轻吸气 → 眨眼 → 睁眼放松 → 微小呼气 → 回到稳定。 |
| `running-right` | 8 | 朝屏幕右侧的轻快小跑。 | 两次交替步态；上身微前倾、编绳与裙摆向后轻微滞后。 |
| `running-left` | 8 | 朝屏幕左侧的轻快小跑。 | 与右跑同样的完整八相步态，朝向与受力方向反转。 |
| `waving` | 4 | 轻抬手的含蓄招呼。 | 手在脸侧中位 → 小幅向右摆 → 回中位 → 小幅向左摆，回到起势。 |
| `jumping` | 5 | 有准备与缓冲的轻跳。 | 轻蹲蓄力 → 刚离地 → 温和顶点 → 回落预备 → 双脚落地回稳。 |
| `failed` | 8 | 克制的受挫与恢复。 | 低落站姿 → 目光下垂 → 肩线下沉 → 闭眼短暂停顿 → 缓慢恢复。 |
| `waiting` | 6 | 双手靠近领结的耐心等待。 | 手靠近胸前 → 微前倾期待 → 小幅示意 → 安静停顿 → 放松 → 回到起势。 |
| `running` | 6 | 专注处理任务的无道具手部节奏。 | 集中视线 → 双手向前 → 交替短促操作 → 扫视 → 回到集中姿势。 |
| `review` | 6 | 认真检查后的确认。 | 观察 → 手靠近下巴 → 细看停顿 → 轻点头确认 → 手放下 → 回正。 |

## 跳跃验收重点

`jumping` 的五帧由 Codex 固定时长播放，不能通过更改 FPS 放慢。因此用清晰的蓄力与落地缓冲形成节奏；顶点只是一段温和的小跳，禁止悬浮、橡皮筋拉伸、大小闪烁或横向抖动。第五帧双脚必须回到正常基线，以平稳接回第一帧。

## 完成顺序

1. 生成并确认身份主形象。
2. 先制作 `idle` 与 `running-right`，确认脸、比例、步态和小尺寸可读性。
3. 制作余下七个标准动作；每一条带单独抽帧和循环检查。
4. 制作四个方向锚点及两行连续观察方向。
5. 组装 8×11 V2 图集、去绿、质检、预览并打包安装文件。
